## Admission Atlas — Armenia

Admission Atlas is an interactive data visualization platform that structures and visualizes bachelor admission data from Armenian universities.

The platform aggregates data from more than 20 official sources and presents them through interactive visualizations to help applicants better understand the admission process and make data-driven decisions.
